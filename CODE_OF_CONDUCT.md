# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/), version 2.1.

In short: be respectful, assume good faith, no harassment. We want a community that argues about ideas, not people.

## Enforcement

Report unacceptable behavior to conduct@openadlc.com. Reports are handled confidentially. Maintainers who do not uphold the code in good faith may face consequences determined by the project leads.
