# Support

- **Docs:** start in [`docs/`](docs/) (and openadlc.com once it is live).
- **Questions and ideas:** open a GitHub Discussion.
- **Bugs:** open a GitHub Issue with steps to reproduce.
- **Security:** see [SECURITY.md](SECURITY.md). Never file a security problem publicly.

OpenADLC is early. Response times are best-effort.
