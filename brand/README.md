# OpenADLC brand

The mark is `/O`: the forward slash is the harness (every OpenADLC action is a command), the `O` is OpenADLC. Amber on black.

## Tokens

- Amber (brand / action): `#EBA836`
- Cream (text on dark): `#F4F1EA`
- Black (canvas): `#000000`
- Type: Martian Mono (display / wordmark), Inter (body)

## Files

- `openadlc-mark.svg`: the `/O` mark, amber on transparent. Primary.
- `openadlc-tile.svg`: app icon, amber `/O` on a black rounded tile (256).
- `favicon.svg`: tighter tile for favicon use.

PNG exports (16 / 32 / 180 / 512 / 1024) derive from these SVGs.

## Wordmark lockup

The `/O` tile next to "OpenADLC" set in Martian Mono: "Open" in cream, "ADLC" in amber.

## Rules

- Amber is the only accent, used sparingly. Never large amber fills.
- The mark is vector; the slash and `O` share one cap height. Do not rebuild it from live font text (the slash overshoots cap height).
- On light backgrounds keep the black tile; the mark is amber-on-black.
